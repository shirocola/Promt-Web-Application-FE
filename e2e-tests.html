Test Suite 3: Employee Form

TC-FORM-001: Form Display

Navigate to Add Employee page
Verify all form fields are present
Check field labels are correct
Verify required field indicators (*)
Check submit button is present
Expected Result: Form displays with all required fields: First Name*, Last Name*, Email*, Position*, Department, Salary, Hire Date

TC-FORM-002: Valid Form Submission

Fill in all required fields with valid data:
• First Name: "Test"
• Last Name: "User"
• Email: "test.user@example.com"
• Position: "Test Engineer"
• Department: "QA"
• Salary: "60000"
• Hire Date: Current date
Click Submit button
Expected Result: Form submits successfully, shows success message, and clears the form

TC-FORM-003: Form Validation

Try submitting form with empty required fields
Enter invalid email format
Enter single character in name fields
Enter negative salary
Enter future date for hire date
Expected Result: Appropriate validation errors are displayed for each invalid field

TC-FORM-004: Duplicate Email Validation

Try to create employee with email: "john.doe@example.com"
Fill other required fields with valid data
Submit the form
Expected Result: Error message "Employee with this email already exists" is displayed
